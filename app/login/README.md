# ログインディレクトリ 🔑

このディレクトリはユーザー認証関連のコンポーネントとページを含んでいます。

## 📑 主要機能

- ユーザーログイン
- アカウント登録
- パスワードリセット
- OAuth認証（Google、GitHubなど）

## 🔑 認証フロー

1. ユーザーがログインページにアクセス
2. 認証情報を入力または外部プロバイダーを選択
3. 認証成功後、ダッシュボードへリダイレクト

## 📌 注意点

- 認証ロジックは`lib/auth.ts`に実装されています
- セキュリティ対策としてRate Limitingを実装しています
- 認証状態はReact Context APIを使用して管理しています
