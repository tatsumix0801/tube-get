# チャンネル分析ディレクトリ 📊

このディレクトリはYouTubeチャンネルの詳細分析機能を提供するページを含んでいます。

## 📑 主要機能

- パフォーマンス分析ダッシュボード
- 視聴者統計と傾向
- コンテンツ最適化提案
- 競合チャンネル比較

## 🔑 ページ構成

- `page.tsx` - 分析ホームページ
- `[id]/page.tsx` - 特定チャンネルの分析
- `[id]/audience/page.tsx` - 視聴者分析
- `[id]/content/page.tsx` - コンテンツ分析
- `[id]/comparison/page.tsx` - 競合比較

## 📈 データ可視化

- インタラクティブなグラフとチャート
- 時系列データの表示
- カスタムレポート生成
- データエクスポート機能

## 📌 注意点

- データ可視化には`components/ui/chart.tsx`を使用してください
- 大量のデータ処理はサーバーサイドで行い、結果のみクライアントに送信してください
- エクスポート機能は`lib/export.ts`を活用してください
