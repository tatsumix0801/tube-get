# チャンネルディレクトリ 📺

このディレクトリはYouTubeチャンネル情報表示と管理のためのページを含んでいます。

## 📑 主要機能

- チャンネル概要表示
- チャンネルの基本情報管理
- 動画一覧表示
- サブスクライバー統計

## 🔑 ページ構成

- `page.tsx` - チャンネル一覧/概要ページ
- `[id]/page.tsx` - 個別チャンネル詳細ページ
- `[id]/videos/page.tsx` - チャンネル動画一覧
- `[id]/settings/page.tsx` - チャンネル設定

## 📊 データ取得

- チャンネルデータはYouTube Data APIから取得
- 認証済みユーザーのチャンネルのみアクセス可能
- サーバーサイドレンダリングとクライアントサイドの状態管理を組み合わせて最適化

## 📌 注意点

- API呼び出しには`lib/youtube-api.ts`のユーティリティ関数を使用してください
- チャンネルIDはURLパラメータから取得します
- 未認証ユーザーはログインページへリダイレクトされます
