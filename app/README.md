# アプリケーションディレクトリ 📱

このディレクトリはNext.jsのApp Routerを使用した主要なアプリケーションコードを含んでいます。

## 📂 フォルダ構成

- `api/` - バックエンドAPIエンドポイント
- `channel/` - チャンネル詳細ページ
- `channel-analysis/` - チャンネル分析ページ
- `dashboard/` - ダッシュボード画面
- `docs/` - ドキュメントページ
- `faq/` - よくある質問ページ
- `login/` - ログイン画面
- `logout/` - ログアウト画面
- `settings/` - 設定画面
- `styleguide/` - スタイルガイドページ

## 🔑 主要ファイル

- `layout.tsx` - アプリケーション全体のレイアウト
- `page.tsx` - メインランディングページ
- `globals.css` - グローバルCSSスタイル

## 📌 注意点

- 各ルートディレクトリには、`page.tsx`ファイルが必要です
- ルート固有のレイアウトは`layout.tsx`で定義してください
- APIルートは`api/`ディレクトリ内に作成してください
