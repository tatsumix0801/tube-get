# ダッシュボードディレクトリ 📊

このディレクトリはユーザーの主要なダッシュボード画面を提供するページを含んでいます。

## 📑 主要機能

- チャンネル概要カード
- 重要指標のクイック表示
- 最近の動画パフォーマンス
- アラートと通知
- パーソナライズされたレコメンデーション

## 🔑 ページ構成

- `page.tsx` - メインダッシュボード
- `stats/page.tsx` - 詳細統計ページ
- `alerts/page.tsx` - アラート管理
- `recommendations/page.tsx` - レコメンデーション

## 🖥️ ユーザーインターフェース

- レスポンシブで直感的なUIデザイン
- ドラッグ＆ドロップでカスタマイズ可能なウィジェット
- ダークモード対応
- モバイルフレンドリーなレイアウト

## 📌 注意点

- ダッシュボードのウィジェットは`components/ui/`のコンポーネントを活用してください
- データフェッチングはReact Query（TanStack Query）を使用してキャッシュを最適化してください
- ユーザー設定は`lib/user-settings.ts`で管理してください
