# FAQディレクトリ ❓

このディレクトリはよくある質問（FAQ）とその回答を提供するページを含んでいます。

## 📑 主要機能

- カテゴリ別のよくある質問
- 検索機能
- 関連質問の提案
- 新しい質問の提出フォーム
- 最終更新日表示

## 🔑 ページ構成

- `page.tsx` - FAQ一覧ページ
- `[category]/page.tsx` - カテゴリ別FAQページ
- `search/page.tsx` - FAQ検索ページ
- `submit/page.tsx` - 質問提出ページ

## 🔍 コンテンツ構造

- 質問と回答のペア
- カテゴリタグ付け
- アコーディオン形式での表示
- バージョン情報と最終更新日

## 📌 注意点

- FAQコンテンツは定期的に更新し、最新情報を反映させてください
- アコーディオンコンポーネントには`components/ui/accordion.tsx`を使用してください
- ユーザーからの質問は管理者がレビューしてから公開してください

## 📋 コンテンツ監査ガイドライン

### 監査頻度
- 重要項目: 隔週レビュー
- 一般項目: 月次レビュー
- 全項目: 四半期レビュー

### 監査ポイント
1. **事実確認**
   - 回答内容が最新の仕様と一致しているか
   - リンク先が有効か
   - スクリーンショットやUIの説明が現行バージョンと一致しているか

2. **完全性チェック**
   - 質問に対する回答が完全か
   - 必要な情報が漏れていないか
   - 追加すべき補足情報はないか

3. **明確性チェック**
   - 説明が明確でわかりやすいか
   - 専門用語の適切な説明があるか
   - 手順が具体的かつ順序立てられているか

4. **コンテンツ関連性チェック**
   - 現在のサービス内容に合致しているか
   - 古くなった情報や機能説明がないか
   - よく検索されるが回答がない質問の洗い出し

### 更新手順
1. 監査シートに対象FAQアイテムをリストアップ
2. 各項目をレビューし、必要な更新内容を記録
3. 優先度に基づき更新を実施
4. 更新日を記録し、タグを更新
5. 次回レビュー日を設定

## 🔄 コンテンツ更新プロセス

### 新規FAQの追加
```tsx
// page.tsxファイル内の該当カテゴリにアイテムを追加
{
  question: "新しい質問タイトル",
  answer: [
    "回答の詳細をここに記載します。",
    "複数行の場合は配列で記述します。",
    "リンクや強調表示も含められます。"
  ],
  lastUpdated: "YYYY年MM月DD日",
  popular: false // 人気質問とする場合はtrue
}
```

### FAQ更新チェックリスト
- [ ] 事実関係の正確性を確認
- [ ] リンクの有効性をチェック
- [ ] 表現の明確さと簡潔さを確認
- [ ] 最終更新日を更新
- [ ] 必要に応じて「人気」タグを調整
- [ ] カテゴリの適切さを確認

```

### 更新項目
``` 
