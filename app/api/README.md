# APIディレクトリ 🔌

このディレクトリはNext.jsのAPIルートを使用したバックエンドエンドポイントを含んでいます。

## 📡 API構造

- RESTful APIエンドポイント
- サーバーサイドの処理ロジック
- 外部APIとの連携（YouTube Data API等）

## 🔑 主要エンドポイント

- `/api/auth/*` - 認証関連
- `/api/channels/*` - チャンネル情報
- `/api/videos/*` - 動画データ
- `/api/analytics/*` - 分析データ
- `/api/export/*` - データエクスポート

## 📊 データフロー

1. クライアントからのリクエスト受信
2. 必要に応じて外部APIにリクエスト
3. データ処理と整形
4. レスポンス返却

## 📌 注意点

- すべてのAPIは適切なエラーハンドリングを実装してください
- リクエスト検証にはzodを使用してください
- 認証が必要なエンドポイントには適切なミドルウェアを適用してください
