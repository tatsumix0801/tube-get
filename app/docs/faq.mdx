# よくある質問 (FAQ)

つべナビに関するよくある質問と回答をまとめています。

## 基本的な質問

### Q: つべナビの利用を開始するには何が必要ですか？

A: 以下の手順で始めることができます：

1. アカウント登録（メールアドレスとパスワード）
2. YouTube Data APIキーの取得（[設定方法ガイド](/docs/api-key-guide)を参照）
3. APIキーをプロフィール設定に登録

このガイドでは、Google Cloudアカウントでのプロジェクト作成から、YouTube Data APIの有効化、APIキーの作成、つべナビでの設定まで詳しく解説しています。

## 分析機能について

### Q: どのような指標で分析できますか？

A: つべナビでは以下の指標を用いて動画のパフォーマンスを分析します。
- 視聴回数
- 視聴時間
- エンゲージメント率（いいね数、コメント数、共有数など）
- クリック率
- 視聴者維持率
- 登録者獲得数

これらの指標を組み合わせて総合的なパフォーマンススコアを算出しています。

### Q: データはどのくらいの頻度で更新されますか？

A: データは基本的に24時間ごとに更新されます。最新のデータを確認したい場合は、画面右上の更新ボタンをクリックすることで手動更新も可能です。

### Q: レポートの出力形式は？

A: Excel形式でのデータ出力に対応しています。レポートには選択した期間のデータ、グラフ、主要な指標のサマリーが含まれます。

## トラブルシューティング

### Q: 「APIキーが無効です」というエラーが表示されます

A: 以下の点をご確認ください。
1. API Keyが正確にコピーされているか
2. YouTube Data API v3が有効化されているか
3. API Keyに適切な制限が設定されているか
4. 使用量制限に達していないか

詳細は[API Keyトラブルシューティング](/docs/api-key-guide#トラブルシューティング)をご覧ください。

### Q: データが表示されない、または一部のデータのみ表示されます

A: 以下の原因が考えられます。
1. チャンネルのプライバシー設定で統計情報が非公開になっている
2. チャンネルや動画が最近作成されたもので、データがまだ十分に蓄積されていない
3. API制限に達している

設定を確認し、時間をおいて再度お試しください。問題が解決しない場合はサポートまでご連絡ください。

### Q: アカウントにログインできません

A: 以下の手順をお試しください。
1. ブラウザのキャッシュとCookieをクリア
2. パスワードリセット機能を使用
3. 別のブラウザでアクセス

これらの方法で解決しない場合は、サポートまでご連絡ください。

## セキュリティ・プライバシー

### Q: 個人情報は安全に保護されていますか？

A: はい。つべナビは必要最小限の権限のみを要求し、パスワードなどの機密情報を保存することはありません。すべての通信はSSL暗号化され、データはセキュアなサーバーで管理されています。詳しくは[プライバシーポリシー](/docs/privacy-policy)をご覧ください。

### Q: どのようなデータが保存されますか？

A: チャンネル情報、動画統計、分析結果、ユーザーアカウント情報などが保存されます。個人を特定できる視聴者データは収集・保存しません。詳細は[プライバシーポリシー](/docs/privacy-policy)をご参照ください。
