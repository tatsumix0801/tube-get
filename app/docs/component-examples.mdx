## 基本UIコンポーネント

### Button

基本的なボタンコンポーネントです。複数のバリエーションがあります。

```tsx
import { Button } from "@/components/ui/button";

// 基本的な使用法
<Button>クリック</Button>

// バリエーション
<Button variant="default">デフォルト</Button>
<Button variant="destructive">削除</Button>
<Button variant="outline">アウトライン</Button>
<Button variant="ghost">ゴースト</Button>
<Button variant="link">リンク</Button>

// サイズ
<Button size="default">デフォルト</Button>
<Button size="sm">小</Button>
<Button size="lg">大</Button>

// 無効状態
<Button disabled>無効</Button>

// アイコンつき
import { FileDownload } from "lucide-react";
<Button>
  <FileDownload className="mr-2 h-4 w-4" />
  ダウンロード
</Button>
```

### Card

情報を表示するためのカードコンポーネントです。

```tsx
import {
  Card,
  CardContent,
  CardDescription,
  CardFooter,
  CardHeader,
  CardTitle,
} from "@/components/ui/card";

<Card>
  <CardHeader>
    <CardTitle>チャンネル統計</CardTitle>
    <CardDescription>チャンネルの主要な指標</CardDescription>
  </CardHeader>
  <CardContent>
    <p>コンテンツをここに配置</p>
  </CardContent>
  <CardFooter>
    <p>フッターコンテンツ</p>
  </CardFooter>
</Card>
```

### Tabs

タブ切り替えインターフェースを提供するコンポーネントです。

```tsx
import { Tabs, TabsContent, TabsList, TabsTrigger } from "@/components/ui/tabs";

<Tabs defaultValue="overview" className="w-full">
  <TabsList>
    <TabsTrigger value="overview">概要</TabsTrigger>
    <TabsTrigger value="analytics">分析</TabsTrigger>
    <TabsTrigger value="reports">レポート</TabsTrigger>
  </TabsList>
  <TabsContent value="overview">概要の内容...</TabsContent>
  <TabsContent value="analytics">分析の内容...</TabsContent>
  <TabsContent value="reports">レポートの内容...</TabsContent>
</Tabs>
```

## 機能コンポーネント

### VideoTable

動画一覧を表示するテーブルコンポーネントです。

```tsx
import { VideoTable } from "@/components/video-table";

// 基本的な使用法
<VideoTable videos={videos} />

// 表示設定を使用する場合
<VideoTable 
  videos={videos} 
  displaySettings={{
    showViews: true,
    showLikes: true,
    showComments: true,
    showSpreadRate: true,
    showDuration: true,
    showTags: true
  }} 
/>

// カスタムアクション
<VideoTable 
  videos={videos} 
  onVideoClick={(video) => {
    console.log("動画がクリックされました:", video);
  }}
/>
```

### ChannelImageDownload

チャンネルのアイコンとバナー画像をダウンロードするためのコンポーネントです。

```tsx

<ChannelImageDownload

  apiKey="YOUR_YOUTUBE_API_KEY"
/>
```

### ChannelTags

チャンネルのタグ（キーワード）を表示するコンポーネントです。

```tsx
import { ChannelTags } from "@/components/channel-tags";

// タグのリストを直接渡す場合
<ChannelTags tags={["ゲーム", "エンタメ", "テクノロジー"]} />

// APIから自動取得する場合
<ChannelTags
  channelId="UCxxxxxxxxxxxxxxxx"
  apiKey="YOUR_YOUTUBE_API_KEY"
  autoFetch={true}
/>
```

### VideoTags

動画のタグを表示および管理するコンポーネントです。

```tsx
import { VideoTags } from "@/components/video-tags";

// 読み取り専用モード
<VideoTags
  videoId="video123"
  tags={["ゲーム", "実況", "攻略"]}
  readonly={true}
/>

// APIから自動取得するモード
<VideoTags
  videoId="video123"
  apiKey="YOUR_YOUTUBE_API_KEY"
  autoFetch={true}
/>

// 編集可能モード
<VideoTags
  videoId="video123"
  tags={initialTags}
  readonly={false}
  onTagsChange={(newTags) => {
    console.log("タグが更新されました:", newTags);
  }}
/>
```

## 状態管理との連携

各コンポーネントは状態管理と組み合わせて使用することができます。以下はReact Hooksを使った例です。

```tsx
import { useState } from "react";
import { DisplaySettings } from "@/components/display-settings";
import { VideoTable } from "@/components/video-table";

function VideoAnalyticsPage() {
  const [displaySettings, setDisplaySettings] = useState({
    showViews: true,
    showLikes: true,
    showComments: true,
    showSpreadRate: true,
    showDuration: true,
    showTags: false
  });
  
  return (
    <div>
      <DisplaySettings
        initialSettings={displaySettings}
        onChange={setDisplaySettings}
      />
      
      <VideoTable
        videos={videos}
        displaySettings={displaySettings}
      />
    </div>
  );
}
```

## カスタマイズ

コンポーネントはテーマやスタイルをカスタマイズできます。Tailwind CSSクラスを使用して直接スタイルを変更したり、テーマプロバイダーを通じてグローバルなスタイル変更を行うことができます。

```tsx
// 個別のコンポーネントのスタイル変更
<Button className="bg-purple-600 hover:bg-purple-700">
  カスタムボタン
</Button>

// テーマの変更
import { ThemeProvider } from "@/components/theme-provider";

<ThemeProvider
  theme={{
    colors: {
      primary: {
        DEFAULT: "#7c3aed",
        foreground: "#ffffff"
      }
      // ...その他のカラー設定
    }
  }}
>
  <App />
</ThemeProvider>
```

これらのコンポーネントを組み合わせることで、一貫性のある美しいUIを構築することができます。
