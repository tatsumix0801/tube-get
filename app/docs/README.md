# ドキュメントディレクトリ 📚

このディレクトリはアプリケーションのドキュメントとユーザーガイドを提供するページを含んでいます。

## 📑 主要機能

- 機能別ユーザーガイド
- チュートリアルとウォークスルー
- APIドキュメント
- よくあるトラブルシューティング

## 🔑 ページ構成

- `page.tsx` - ドキュメントホーム
- `getting-started/page.tsx` - 入門ガイド
- `api/page.tsx` - API利用ガイド
- `features/page.tsx` - 機能別ガイド
- `troubleshooting/page.tsx` - トラブルシューティング

## 📖 コンテンツ管理

- Markdownベースのコンテンツ
- コード例と実行可能なデモ
- 画像とビデオチュートリアル
- 検索機能

## 📌 注意点

- コンテンツはMarkdownで記述し、mdxを使用して動的コンポーネントを組み込んでください
- コードサンプルは正確に、かつ最新の状態を維持してください
- モバイルユーザーも考慮したレスポンシブデザインを確保してください
