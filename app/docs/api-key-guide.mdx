# YouTube API キー取得ガイド

このガイドでは、つべナビで使用するYouTube Data APIキーの取得方法について説明します。

## 1. Google Cloudアカウントにログイン

1. [Google Cloud Console](https://console.cloud.google.com/)にアクセスします。
2. Googleアカウントでログインします。初めての方は、案内に従ってGoogle Cloudアカウントを作成してください。

## 2. プロジェクトの作成

1. トップページの「プロジェクトを選択」をクリックします。
2. 開いたダイアログで「新しいプロジェクト」をクリックします。
3. プロジェクト名（例: "つべナビ API"）を入力し、「作成」をクリックします。
4. プロジェクトが作成されたら、そのプロジェクトを選択します。

## 3. YouTube Data APIの有効化

1. 左側のメニューから「APIとサービス」→「ライブラリ」をクリックします。
2. 検索バーに「YouTube Data API」と入力し、検索結果から「YouTube Data API v3」をクリックします。
3. 「有効にする」ボタンをクリックして、APIを有効化します。

## 4. APIキーの作成

1. 「APIとサービス」→「認証情報」をクリックします。
2. 「認証情報を作成」ボタンをクリックし、ドロップダウンメニューから「APIキー」を選択します。
3. APIキーが生成されます。このキーをメモしておきます。

## 5. APIキーの制限設定（推奨）

セキュリティ向上のため、APIキーに制限を設定することをお勧めします。

1. 生成されたAPIキーの横にある「編集」ボタンをクリックします。
2. 「APIの制限」セクションで「APIの制限」を選択し、リストから「YouTube Data API v3」のみを選択します。
3. 「保存」ボタンをクリックして設定を適用します。

## 6. 請求の設定（必要に応じて）

YouTube Data APIは無料枠（1日あたり10,000ユニット）がありますが、使用量が多い場合は請求の設定が必要です。

1. 左側のメニューから「お支払い」をクリックします。
2. 案内に従って支払い情報を設定します。

## 7. つべナビでの設定

1. つべナビの設定画面で「YouTube API設定」セクションを開きます。
2. 取得したAPIキーを入力フィールドに貼り付けます。
3. 「保存」ボタンをクリックして設定を完了します。

## 注意事項

- APIキーは秘密情報です。他人と共有したり、公開リポジトリにコミットしたりしないでください。
- YouTube Data APIには1日あたりの使用制限があります。詳細は[Google公式ドキュメント](https://developers.google.com/youtube/v3/getting-started)を参照してください。
- APIキーの不正利用を防ぐため、必ず適切な制限を設定してください。

## トラブルシューティング

APIキーに関する問題が発生した場合：

1. キーが正確にコピーされているか確認します。
2. Google Cloud Consoleで該当するAPIが有効になっているか確認します。
3. APIの使用量制限に達していないか確認します。
4. 引き続き問題が解決しない場合は、[Google Cloud Support](https://cloud.google.com/support)またはつべナビサポートにお問い合わせください。
