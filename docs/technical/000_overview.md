# つべナビ プロジェクト概要

## 目的と概要
つべナビは、YouTubeチャンネル情報取得と基本分析を提供するWebアプリケーションです。ユーザーがYouTubeチャンネルの基本情報や動画リストを簡単に取得し、データの整理や基本的な分析ができるようサポートします。

## 主要機能
- **チャンネル基本情報取得**: チャンネル名、登録者数、総再生回数、チャンネルアイコン、タグなどの情報を収集・表示
- **動画リスト管理**: 動画タイトル、公開日、再生数、高評価数などの一覧表示とソート機能
- **期間別フィルタリング**: 全期間/1週間/1ヶ月/3ヶ月/半年/1年の期間でデータを絞り込み
- **Excelレポート出力**: チャンネル情報と動画データをExcel形式でエクスポート
- **サムネイルダウンロード**: 選択期間内の動画サムネイルを一括ZIP形式でダウンロード
- **レスポンシブUI**: あらゆるデバイスに最適化されたモダンなインターフェース
- **ダークモード対応**: ユーザー好みの表示モードを選択可能

## ターゲットユーザー
- YouTubeコンテンツクリエイター
- チャンネル運営マネージャー
- マーケティング担当者
- コンテンツ制作チーム
