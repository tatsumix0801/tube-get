# ✨💖 つべナビ ブランディング更新計画書 💖✨

## 🌟 概要

「つべナビ」のブランドイメージを一新し、より現代的で魅力的なデザインにアップデートするための計画書です。現在のプレースホルダーロゴやカラースキームを見直し、ユーザーに親しみやすく、かつプロフェッショナルな印象を与えるブランディングを目指します。

## 👀 現状分析

### 現在のブランディング要素
- **ロゴ**: プレースホルダーロゴを使用中（`placeholder-logo.png/svg`）
- **カラースキーム**: モノトーンベースのデザイン
- **フォント**: 基本的なArial, Helvetica, sans-serif
- **デザイン傾向**: シンプルで最小限

### 課題点
- 独自性の欠如
- ブランドの個性が確立されていない
- YouTubeコンテンツ分析ツールとしての特徴が視覚的に表現されていない
- ダークモードとライトモードでの一貫性

## 💡 更新方針

### 1. ロゴデザイン
- **コンセプト**: 動画・分析・ナビゲーションを表現する要素を取り入れる
- **カラー**: YouTubeの赤を想起させつつも、独自性のある配色
- **形状**: シンプルで覚えやすく、スマホアイコンとしても認識しやすいデザイン
- **バリエーション**: 
  - フルロゴ（テキスト + シンボル）
  - シンボルのみ（アイコン用）
  - モノクロバージョン

### 2. カラーパレット
- **プライマリーカラー**: #FF3366（ビビッドなピンク - エネルギッシュで目を引く）
- **セカンダリーカラー**: #3A86FF（鮮やかなブルー - 信頼性と分析を表現）
- **アクセントカラー**: #FFD60A（イエロー - 注目ポイントやハイライト用）
- **ニュートラルカラー**: 
  - ライトモード: #F8F9FA → #212529（白→黒のグラデーション）
  - ダークモード: #212529 → #F8F9FA（黒→白のグラデーション）

### 3. タイポグラフィ
- **見出し**: Montserrat（モダンでクリーン、太字で存在感あり）
- **本文**: Inter（可読性高く、画面表示に最適化）
- **アクセント**: Poppins（特別な要素やCTAボタンに使用）

### 4. デザイン要素
- **アイコンスタイル**: Lucide Reactの一貫したアイコンセット + カスタムアイコン
- **イラスト**: データ可視化をモチーフにした抽象的なイラスト
- **パターン/テクスチャ**: 動画フレームやグラフをモチーフにした背景パターン
- **アニメーション**: 軽快でスムーズな遷移効果
