# 要件定義書 - UI/UXリニューアル & サイト名変更

## 2. 目的

1. **デザインの洗練化**: 参照リポジトリのSF/サイバーパンク風デザインを取り入れ、よりエレガントで先進的なUIに進化させる
2. **サイト名変更**: 「つべナビ」から「TubeVision」へ統一的に変更
3. **ブランドイメージ刷新**: YouTube分析ツールとしての専門性と先進性を視覚的に表現

## 3. デザイン方針

### 3.1 カラーパレット（追加）

```css
/* SF風カラーパレット */
--sf-bg-primary: #0a0a0f;      /* 深い黒 */
--sf-bg-secondary: #0d1117;    /* GitHub風ダークグレー */
--sf-bg-tertiary: #161b22;     /* カード背景 */

--sf-text-primary: #ffffff;
--sf-text-secondary: #c9d1d9;
--sf-text-muted: #8b949e;

--sf-cyan: #00ffff;            /* メインアクセント */
--sf-cyan-dark: #00cccc;       /* ホバー時 */
--sf-magenta: #ff00ff;         /* セカンダリアクセント */
--sf-green: #00ff88;           /* 成功 */
--sf-orange: #ff6b00;          /* 警告 */
--sf-purple: #9945ff;          /* 既存との調和 */

/* ネオンカラー */
--neon-pink: #FF10F0;
--neon-blue: #00D4FF;
--neon-green: #39FF14;
```

### 3.2 デザインエフェクト

| エフェクト | 説明 | 使用箇所 |
|-----------|------|----------|
| **グローエフェクト** | ネオン発光効果 | ボタン、カード、アクセント要素 |
| **グラスモーフィズム** | すりガラス効果 | カード、モーダル、ナビゲーション |
| **サイバーグリッド** | 背景グリッドパターン | ページ背景 |
| **ホログラフィック** | 虹色グラデーションボーダー | 重要カード、ハイライト要素 |
| **スキャンライン** | CRTモニター風効果 | 背景（オプション） |

### 3.3 アニメーション

- `sf-pulse`: 脈動効果（ローディング、アクセント）
- `sf-glitch`: グリッチ効果（エラー、特別なアクション）
- `holo-shift`: ホログラフィック色相変化
- `neon-pulse`: ネオン点滅
- `float`: 浮遊効果（アイコン、装飾要素）

### 3.4 既存デザインとの調和

- 既存の紫/インディゴ系カラー（`--primary: 270°`）を維持
- 新しいSFカラーはアクセントとして追加
- グラデーション効果は既存のものを強化
- ダークモード時にSF効果を強調
