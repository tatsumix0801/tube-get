# スタイルディレクトリ 🎨

このディレクトリはアプリケーション全体で使用されるグローバルスタイルを含んでいます。

## 📑 主要ファイル

- `globals.css` - アプリケーション全体に適用されるグローバルスタイル

## 🔑 構成

グローバルCSSファイルには以下の要素が含まれています：

1. Tailwind CSSディレクティブ（@tailwind base, components, utilities）
2. カスタムCSSプロパティ（色、サイズ、アニメーションなど）
3. 基本的なリセットスタイル
4. フォントの定義
5. カスタムユーティリティクラス

## 🧩 使用例

```css
/* カスタムCSSプロパティの例 */
:root {
  --background: 0 0% 100%;
  --foreground: 222.2 84% 4.9%;
  --primary: 222.2 47.4% 11.2%;
  --secondary: 210 40% 96.1%;
  --accent: 210 40% 96.1%;
}

/* ダークモード対応の例 */
.dark {
  --background: 222.2 84% 4.9%;
  --foreground: 210 40% 98%;
  --primary: 210 40% 98%;
  --secondary: 217.2 32.6% 17.5%;
  --accent: 217.2 32.6% 17.5%;
}
```

## 📌 注意点

- アプリケーション固有のスタイルはTailwind CSSを優先して使用してください
- グローバルCSSは最小限に抑え、コンポーネント固有のスタイルはコンポーネント内で定義してください
- カスタムCSSプロパティの命名は一貫性を保ってください
- メディアクエリはTailwindの設定に合わせて使用してください
- CSSの変更は慎重に行い、意図しない副作用を避けるようにしてください
